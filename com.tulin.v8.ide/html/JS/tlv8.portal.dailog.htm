<static>void openDailog(name, url, width, height, callback, itemSetInit, titleItem, urlParam)
打开对话框

   Parameters:
     name
         {string} 对话框的标题
     url
         {string} 指定要打开的页面
     width
         {string} 对话框宽度
     height
         {string} 对话框高度
     callback
         {string} 回调函数
     itemSetInit
         {string} itemSetInit：对话框按钮
         如item{refreshItem:true,enginItem:true,CanclItem:true}
         refreshItem:刷新按钮 enginItem:确定按钮 CanclItem:取消按钮
    titleItem
         boolean 为false时掩藏标题栏
    urlParam
         {Object} js任意类型可以直接传递到对话框页面 对话框页面通过函数getUrlParam获取
    
         如：function getUrlParam(param){ //此函数在对话框打开时调用

        alert(param);

        }

   Returns:
         {void}

   Example:
         tlv8.portal.dailog.openDailog('人员信息','/system/opm/person-dailog.html',
         500,350,dailogcallback,null)
