# tlv8ide

#### 介绍
这是一款基于eclipse开发的快速开发工具
集成了tomcat插件，数据库插件，报表设计（可视化），流程设计（可视化），网页设计（可视化）等多功能于一体的JAVA EE项目开发工具。 能够实现业务系统快速开发，报表系统快速开发（只需要会SQL），工作流快速定制。
