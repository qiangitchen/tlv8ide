CheckBox tlv8.CheckBox
CheckBox 复选组件

   Parameters:
     div
         {Element div}[required]复选组件所属的层

     item
         {Map}[required]复选组件的选项

     splithtmlarray
         {Array}选项分隔标记数组 内容可以是任意html标签 如"<br>"," " 数组长度与选项个数相同

   Returns:
         {CheckBox}

   Example:
         <div id="sNameDemoMus" ></div>
         var checkboxDemo = document.getElementById("sNameDemoMus");
         var item = new Map();
         item.put("yes","是");
         item.put("no","否");
         item.put("normal","未知");
         var sts = new tlv8.CheckBox(checkboxDemo,item);

      或

         <div id="sNameDemoMus" onchecked="kkkk"></div>
         var checkboxDemo = document.getElementById("sNameDemoMus");
         var item = new Map();
         item.put("yes","是");
         item.put("no","否");
         item.put("normal","未知");
         var splithtmlarray = new Array();
         splithtmlarray.push("&nbsp;&nbsp;&nbsp;&nbsp;");
         splithtmlarray.push("<br>");
         splithtmlarray.push("<br>");
         var sts = new tlv8.CheckBox(checkboxDemo,item,splithtmlarray);
         function kkkk(value){//在DIV标签中注册的kkkk事件不能加括号
           alert(value);
         }
