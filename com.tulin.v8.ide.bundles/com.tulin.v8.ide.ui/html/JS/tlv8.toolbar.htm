         {Element div}[required]用于显示工具栏的DIV标签
     insertitem
         {boolean/string}[required]新增按钮状态("readonly":只读,true：可操作,false：不可见)
     saveitem

     deleteitem

     refreshitem
         {boolean/string}[required]刷新按钮状态("readonly":只读,true：可操作,false：不可见)

   Returns:
         {toolbar}

   Example:

         deleteAction="deleteMaindata()" refreshAction="refreshMainData()"/>
