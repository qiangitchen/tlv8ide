TLv8平台

本平台采用java作为后端逻辑处理语言，
在Eclipse3.5上架构开发自己的产品插件TLv8 IDE
前后端采用AJAX技术进行数据交换，
前端向后台提交数据采用XML数据格式,
后台向前端浏览器发送数据采用JSON数据格式

采用的技术：
前端：DHTML: javascript、html、div+css
后端：JAVAEE: java、spring MVC、mybatis

包含服务:
业务服务(tlv8)、文档服务(daisy)、报表(UReport)、图表(echarts)、消息服务(websocket)

门户：所有服务通过统一门户jpolite管理

网站：www.tlv8.com

源码：gitee.com/qianpou/tlv8ide
